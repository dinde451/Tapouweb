PhpBB

PhpBB est la plateforme open source du forum d'AideDD. C'est celle que j'utilise depuis des années pour le site, mais avec la différence notable que c'est sur la version 3.1 que je me base cette fois-ci. Qu'apporte cette nouvelle version ? Et bien en fait tout le code a été retouché par rapport à la branche 3.0, la 3.1 étant développée en php orienté objet. Et ceci a concrètement deux conséquences pour le site.

La première est que tous les MOD que j'avais développé sur le forum précédent, notamment toutes les balises pour les parties en ligne, sont à refaire. Ce n'est plus compatible, et il me faut tout réapprendre de cette nouvelle version. La deuxième est que la customisation, une fois que je la maîtriserais, devrait être beaucoup plus maintenable, car maintenant on ne modifie plus les fichiers sources du code. Tout comme pour Silverstripe, on développe les extensions (nouveau nom pour les MOD) dans un espace à part. Donc quand une mise à jour de phpBB sort, ça n'affecte plus directement la custom.

Extension Board Rules

C'est la première extension que j'ai installé pour phpBB 3.1. Reprise telle quelle, je ne l'ai pas retouchée (elle n'est donc pas développée par moi). Elle permet de gérer la charte du forum.

Extension New BBCode

Là c'est différent. J'ai pris comme base une extension existante relativement complète et complexe, et je l'ai dépurée fortement. La balise SPOILER est reprise de cette extension par exemple. Et sur cette base je greffe mon propre développement, pour proposer par exemple les balises DICE et SECRET.

Fichier modifié :
posting.php
-> ajout d'un événement pour pouvoir modifier un texte dans la fenêtre d'édition.

Extension Header

Une extension aidedd toute simple qui ne fait qu'insérer du code html dans le template overall_header. Je l'utilise pour ajouter mon favicon, le code de Piwik, le code d'Adsense et le lien vers les règles de D&D 5.

Extension Avatar

Une extension aidedd qui montre quels sont les avatars déjà utilisés par d'autres membres, dans le cas d'avatars de galeries, comme sur AideDD.

Fichiers modifiés :
includes/ucp/ucp_profile.php 
-> ajout d'un événement pour pouvoir comparer l'avatar.

phpbb/avatar/driver/local.php
-> ajout de code pour griser l'avatar.

Extension Maps

Une extension aidedd qui permet de convertir les localisations renseignées par les membres en coordonnées, pour ensuite les afficher sur une carte Google Maps. Le tout de manière automatique, déclenché par une tâche Cron toutes les 24 heures, et gérable depuis le panneau d'administration du forum.

Fichier modifié :
phpbb/profilefields/manager.php
-> ajout de code pour reseter les coordonnées lors de la modification d'un profil.

Extension RPG

Une extension aidedd pour les parties en ligne. Tout d'abord une interface sur le panneau d'administration du forum permet d'assigner automatiquement une partie (création du groupe, assignation des droits, etc). Côté utilisateurs, un nouvel onglet dans le panneau d'utilisateur permet d'accéder à certaines informations qui sont ensuite affichées dans le profil à côté des posts, ainsi qu'à la gestion des battlemaps.
