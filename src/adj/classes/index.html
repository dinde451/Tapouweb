Saveurs de classes

Cette rubrique présente diverses aides de jeu pour donner du relief à la classe de votre personnage ou complémenter les choix possibles.

  • Quelques idées pour sortir des sentiers battus concernant les barbares.
  • Un nouveau collège bardique pour les bardes.
  • Un nouveau domaine divin pour les clercs.
  • Un nouveau cercle druidique pour druides.
  • Une nouvelle table d'effets pour le Sursaut de magie sauvage des ensorceleurs.
  • Tout ce que vous devez savoir sur le rituel qui permet de transformer un magicien en liche.
  • Présentation de la nécromancie, indispensable pour tout magicien nécromant voulant se spécialiser dans cette école.
  • Des idées de pactes pour les occultistes et de nouveaux patrons.
  • Un nouvel archétype pour rôdeurs.
